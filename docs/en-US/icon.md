# Icon

Simply beautiful icons.

::: example

```meta
title: Basic
subtitle: Use `type` to define Icon's style.
```

```js
ReactDOM.render(
  <div className="icon-list clearfix">
    <div className="icon-item">
      <Icon type="home" />
      <span>home</span>
    </div>
  </div>,
mountNode);
```
